# Glow | Editorial Makeup 💄✨

A high-end Landing Page designed for professional makeup artists and editorial services. This project combines a minimalist and luxurious aesthetic with a robust **Django** backend.

## 🚀 Live Demo
You can see the live site here: 
[![Deploy on Render](https://img.shields.io/badge/Render-Live%20Demo-ff69b4?style=for-the-badge&logo=render&logoColor=white)](https://landing-page-1-bieb.onrender.com)

---

## 🛠️ Tech Stack

### Backend
* **Python / Django**: Server-side logic and data management.
* **Gunicorn**: WSGI HTTP Server for production.
* **WhiteNoise**: Efficient static file management on Render.
* **Pillow**: Image processing for the portfolio gallery.

### Frontend
* **Tailwind CSS**: Modern and responsive styling via CDN.
* **Animate On Scroll (AOS)**: Elegant entrance animations.
* **Custom CSS3**: Glassmorphism effects and infinite text marquee animations.

---

## 🌟 Key Features
- **Editorial Design**: Hand-picked typography (`Cormorant Garamond`) for a luxury feel.
- **Before/After Slider**: Interactive comparison for makeup transformations.
- **Responsive Design**: Fully optimized for mobile and desktop devices.
- **Contact Form**: Integrated with Django Admin for client lead management.

2. Create a virtual environment and install dependencies:
   ``bash
   pip install -r requirements.txt
3. Run migrations and start the server:
   ``bash
   python manage.py migrate
   python manage.py runserver
